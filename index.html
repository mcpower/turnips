Turnip price pattern calculator for New Horizons

This work would not be possible without @_Ninji's reverse engineering efforts. This page is based entirely off of their reverse engineering. Thank you, Ninji!

It is highly recommended that you become familiar with the turnip price patterns before using the calculator. Check out the info page for a summary, /u/Edricus's in-depth write-up for a detailed overview, or the decompiled code for the gory details.

Put your known prices in below and the calculator will predict the prices based on the patterns above. You can leave any price blank.

Some other people have also made calculators based on Ninji's reverse engineering: mikebryant, elxris. Be sure to check them out!

This calculator is a work in progress! While the output for the calculator is very primitive and not pretty, the calculations should be correct.
